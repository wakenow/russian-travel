# Проект 3: Путешествия по России
## Описание проекта
Третья проектная работа онлайн-курсов Яндекс.Практикума.
Одностраничный сайт, созданный по правилам Nested БЭМ. Выполнен с использованием flexbox и grid-layout. Используется "резиновая" вёрстка для популярных разрешений.
## Планы по доработке
1. Проверить код на кроссбраузерность и дописать все вендорные префиксы.
2. Реализовать более плавное сжатие сайта на разных устройствах.
